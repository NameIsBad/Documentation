**HTTP状态码**
==============

+--------+-----------------------+-----------------------------+
| 状态码 | 状态码英文名称        | 说明                        |
+========+=======================+=============================+
| 200    | OK                    | 请求成功。                  |
+--------+-----------------------+-----------------------------+
| 400    | Bad Request           | 请求错误。{ “errorCode”: 0, |
|        |                       | “message”: “错误信息” }     |
+--------+-----------------------+-----------------------------+
| 401    | Unauthorized          | 请求要求用户的身份认证      |
+--------+-----------------------+-----------------------------+
| 500    | Internal Server Error | 服                          |
|        |                       | 务器内部错误，无法完成请求  |
+--------+-----------------------+-----------------------------+
