**office文档、图片转pdf文档**
==================================

**请求地址**

::

   POST https://transapi.dl.wiki/conversions/file-to-pdf

**请求参数**

==== ====== ====== ==== ============
属性 类型   默认值 必填 说明
==== ====== ====== ==== ============
file binary        是   转换文件(流)
==== ====== ====== ==== ============

**返回值**

====== ======
类型   说明
====== ======
stream 文件流
====== ======

*Response Heads(响应头信息)*

================= ====== ============================================================
属性              类型   说明
================= ====== ============================================================
X-User-TotalPages int    剩余的总页数
X-User-Pages      int    当前使用的页数
================= ====== ============================================================
